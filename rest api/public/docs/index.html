Authentication

Login

POST /api/login

Request Body:

{
    "email": "user@example.com",
    "password": "password123"
}

Response:

{
    "status": 200,
    "token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."
}

Register

POST /api/register

Request Body:

{
    "name": "John Doe",
    "email": "john@example.com",
    "password": "password123"
}

Response:

{
    "status": 201,
    "message": "User created successfully",
    "id": 1
}

Users

List Users

GET /api/users

Headers required:

Authorization: Bearer {token}

Response:

[
    {
        "id": 1,
        "name": "John Doe",
        "email": "john@example.com",
        "created_at": "2024-01-10 12:00:00",
        "updated_at": "2024-01-10 12:00:00"
    }
]

Sliders

Create Slider

POST /api/sliders

Headers required:

Authorization: Bearer {token}
Content-Type: multipart/form-data

Request Body:

title: "Main Banner"
description: "Welcome to our company"
image: (file)

Response:

{
    "id": 1,
    "title": "Main Banner",
    "description": "Welcome to our company",
    "image": "1705123456_image.jpg",
    "created_at": "2024-01-10 12:00:00"
}
